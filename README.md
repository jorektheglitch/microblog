# Microblog

## Setup

``` batch
git clone https://github.com/jorektheglitch/microblog
cd ./microblog
python -m venv ./.venv
./.venv/Scripts/activate
pip -r requirements.txt
./.venv/Scripts/deactivate
```

## Launch

### Windows

``` batch
run
```

### Linux

``` bash
./.venv/Scripts/activate
flask run
```
